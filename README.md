# electerm
ssh/sftp client(mac, win, linux) based on electron/node-pty/xterm...

## dev
```
# with node 8.6+

git clone git@github.com:electerm/electerm.git
cd electerm
npm i

# server
npm run s

# client
npm run c

# app
npm run dev
```
